### New Backend Capabilities
- `GET /caredesk/files?path={storagePath}` -- authenticated file serving for evidence and checklist images.
- `CaredeskNasStorageAdapter.read()` -- reads binary files from `.data/caredesk-nas` with path traversal guard.
- `POST /caredesk/reports/export-csv` -- Owner-only CSV export of jobs with date-range filter.
- `POST /caredesk/reports/export-pdf` -- Owner-only PDF report export via Puppeteer.
- Daily cron (`@nestjs/schedule`) at midnight -- pickup reminder scheduler (Day 0/7/14/30/60) and auto-transition to `UNCLAIMED` at configured `unclaimedDay`.
- Weekly data retention cron -- auto-purges completed jobs older than configured retention days.
